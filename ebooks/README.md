# EBS 교재 EPUB 폴더

YES24에서 다운로드한 입트영/귀트영 월간 교재 EPUB 파일을 여기에 넣어주세요.

## 파일명 규칙
- `입트영_2026-04.epub` (4월호)
- `귀트영_2026-04.epub` (4월호)

## 사용 방법
1. YES24 Crema 앱에서 교재 다운로드
2. EPUB 파일을 이 폴더에 복사
3. generate_daily.py 실행 시 자동으로 해당 날짜 콘텐츠를 추출합니다

## 콘텐츠 우선순위
1. EPUB 교재 (가장 정확)
2. source/ 폴더의 텍스트 파일
3. EBS 에피소드 제목 (자동 조회)
4. Claude 자체 생성 (폴백)
